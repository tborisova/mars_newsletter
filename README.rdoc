== Mars Newsletter

* Usage
  * register user:
  
   POST /auth
   params: email, password, password_confirmation
  
  * sign in:
  
   POST /auth/sign_in
   params: email, password
  
  * create entry for newsletter
  
   POST /entries
   params: entry[title], entry[body]
   headers: access-token, uid, client
  
  * fetch all newsletters:
   GET /newsletters
